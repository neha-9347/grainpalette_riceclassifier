About This Project

GrainPalette is a deep learning-based web application that classifies rice types using transfer learning. The model was trained on a variety of rice grain images including Basmati, Arborio, Jasmine, Karacadag, and Ipsala. It uses MobileNetV2 architecture for accurate predictions with minimal latency.

This project showcases the power of AI in agriculture, helping automate and improve grain quality detection.
